# Presentation — "Grading the Trace" (Part 2)

A self-contained 7-slide deck for the 5–7 minute cohort share-out. No build, no
dependencies, no network needed — just open the HTML file.

**File:** [`index.html`](index.html)

## Present it
1. Open `index.html` in any modern browser (double-click, or drag it into a tab).
2. Navigate: **→ / Space / click** = next · **← / click far-left** = previous · **Home/End** = first/last.
3. **F** = fullscreen · **P** = print/PDF. A progress bar + slide counter sit at the bottom.

Designed at 16:9; it scales to fill whatever screen you present on.

## Make a PDF (for Canvas upload)
The deck has print styling that puts **one slide per landscape page**:
1. Open `index.html`, press **Ctrl/Cmd + P**.
2. Destination → **Save as PDF**; Layout → **Landscape**; Margins → **None**; (enable "Background graphics").
3. Save as `Grading-the-Trace.pdf`.

*(No headless browser is installed in this environment, so the PDF is produced from your browser — it renders the deck faithfully, which LibreOffice would not.)*
