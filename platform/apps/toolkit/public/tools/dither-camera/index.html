What is Dithering?

Dithering converts a photograph with continuous shades of gray into a pattern of pure black and white marks. This is essential for PIAF tactile printing, which can only raise or not raise the paper — there are no in-between tones. Different dithering algorithms create different visual textures while preserving the impression of the original image's light and dark areas.

Algorithms

Dots — Variable-size shapes on a regular grid. Darker areas get bigger shapes, lighter areas get smaller ones. Choose from circle, cross, dash, or diamond shapes. Good general-purpose choice for tactile printing because the dot spacing is predictable and easy to feel.

Floyd — Floyd-Steinberg error diffusion. Each pixel is rounded to black or white, and the rounding error is spread to neighboring pixels. Produces detailed, organic-looking results that closely follow the original image. Can feel noisy on tactile prints at fine scales.

Atkinson — Bill Atkinson's error diffusion, made famous on the original Macintosh. Like Floyd-Steinberg but intentionally discards some error, creating a lighter, more open result with more white space. Better for tactile printing when you want less clutter.

Bayer — Ordered dithering using a 4x4 threshold matrix. Creates a regular, grid-like pattern. Has a distinctive retro/digital look. The structured pattern can be easier to read by touch because it's highly regular.

Halftone — Diamond-shaped dots on a 45-degree rotated grid. Dots grow with darkness and merge into a connected diamond mesh at mid-tones. This is the classic amplitude-modulated screen used in newspaper and magazine printing. Very distinct tactile texture.

Lines — Horizontal bars whose thickness varies with darkness. Light areas have thin lines, dark areas have thick bars that merge together. Simple and effective for architectural drawings and graphics with clear horizontal features.

Hatch — Cross-hatching that layers lines at different angles based on darkness. Light areas get a single horizontal line, mid-tones add vertical, darker areas add diagonals. Mimics traditional pen-and-ink illustration technique. Rich tactile texture.

Engrave — Continuous diagonal lines with variable thickness. Produces the look of engraved banknotes or etched illustrations. The lines follow a consistent 45-degree angle but swell and thin with the image's tones.

Stipple — Randomly placed dots with density proportional to darkness. Unlike the regular grid of Dots, stipple places dots at random positions, creating a hand-drawn pointillist effect. Uses a deterministic random seed so the same image always produces the same output.

OX — Uses two distinct mark shapes: O (circle outlines) for lighter areas and X (diagonal crosses) for darker areas. Mid-tones overlay both. Creates a tactile texture where you can feel the difference between light and dark by mark shape, not just density.

Controls

Spacing / Scale — Controls how large the dithering pattern is. Higher values produce bigger, chunkier marks with more space between them. Lower values produce finer detail. For tactile printing, larger spacing (10-16) is usually better.

Gamma — Adjusts the midtone balance. Values below 1.0 darken the image, values above 1.0 lighten it. Useful for bringing out detail in shadows or highlights.

Contrast — Increases or decreases the difference between light and dark areas. Positive values make lights lighter and darks darker. Negative values flatten the tonal range.

Brightness — Shifts all tones lighter (positive) or darker (negative).

Invert — Swaps black and white. Useful when printing on dark paper or when you want raised marks to represent light areas instead of dark.

Color — Enables CMYK color halftone mode. Separates the image into Cyan, Magenta, and Yellow channels, each rendered as diamond halftone dots at classic print screen angles (C=15°, M=75°, Y=0°). Overlapping dots mix subtractively to reproduce the full color spectrum, just like newspaper and magazine color printing.

Edges — Overlays detected edges on top of the dithered image. Uses a Sobel filter to find areas of high contrast (object boundaries, architectural lines) and draws them as thin lines. Works with any algorithm and with color mode. Especially useful for architectural images where you want structural lines to remain crisp.

Blur/Sharp — Pre-processing filter applied before dithering. Negative values blur (soften), positive values sharpen (enhance edges). Useful for reducing noise before dithering or emphasizing detail.

Posterize — Reduces the number of tonal levels before dithering. Low values (2-4) create dramatic banding effects. Set to 32 for off.

Depth — Uses an AI model (Depth Anything V2) to estimate distance from camera. Bright areas are close, dark areas are far. First use downloads ~40MB model (cached afterward). Tap to toggle between depth map and original photo. All algorithms work on the depth data.

Bracket — Saves three versions at once: tight (60% spacing), medium (current), and loose (160% spacing). Handy for testing which spacing works best on your PIAF machine.

Keyboard Shortcuts

Space: Save image. B: Bracket capture. I: Toggle invert. +/-: Adjust spacing. [/]: Adjust gamma.
