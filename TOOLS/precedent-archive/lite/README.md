# Precedent Archive — lite demo

A single, self-contained HTML file that shows the tool with **six real precedents**
(Salk, Kimbell, Exeter, Farnsworth, Barcelona Pavilion, Pantheon) + one un-cataloged
image. No install, no server, no API key — just open it.

```bash
open index.html        # macOS  (or double-click it / drag into a browser)
```

## What's real vs. canned
- **Live in your browser:** browse, search, type/tag filters, gallery/list, entry detail,
  the alt-text toggle, relations (add/remove), **local add** (drag-drop / paste / choose
  files), and — actually computed on `<canvas>` — **PIAF prep** (greyscale → threshold →
  1-bit, optional edge-detect) and **basic edits** (rotate / resize / contrast), both with
  real downloadable PNGs.
- **Pre-baked (a static page can't do these):** *AI alt text* — "Generate" fills a written
  string; the full `web/` tool calls `claude-sonnet-4-6`. *URL fetch* — makes a manual
  stub; the full tool fetches Open Graph / oEmbed server-side.
- **Images:** clean SVG illustrations of each project (offline, license-safe). Real
  metadata and source links are attached to every entry.

## Swapping in the real photographs
The images are generated by a small pipeline that *tries Wikimedia Commons first and falls
back to the illustrations* when offline. On a machine with normal internet:

```bash
cd src
node build.mjs        # fetches each project's Commons photo (+ author/license/source)
                      # and rewrites ../index.html with the real images embedded
```

This environment blocks Wikimedia, so the committed `index.html` ships with the
illustrations; one run elsewhere swaps in the photos.

## Files
- `index.html` — the committed, self-contained demo (open this).
- `src/template.html` — the app (HTML + inline CSS/JS) with a `/*__DATA__*/` slot.
- `src/data.mjs` — the canned archive: real metadata, tuned alt text, relations, Commons refs.
- `src/illustrations.mjs` — the offline SVG stand-ins.
- `src/build.mjs` — zero-dep assembler (Commons → fallback → inject → write `index.html`).
