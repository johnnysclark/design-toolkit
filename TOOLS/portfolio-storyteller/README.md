# Review / Portfolio Storyteller Aid

**Help students turn a project into a story — for the review and for the portfolio.**
Two related jobs: prep for the **review/crit** (anticipate questions, sharpen the
argument, rehearse the narrative) and shape the **portfolio narrative** (project
statements, sequencing, captions) — with the student's *voice* and *judgment* kept
front and center.

**Status:** 🟡 planned / scoping. Maps to `TOOL-CATALOG.md` §13 (Critique) and §15
(Portfolio & Communication) — jury-prep simulator, crit weather report, portfolio
narrative drafter, audience re-voicer, talk-track generator.

## Two modes (candidate)
- **Review aid** — forecast likely critic questions (tagged fair / loaded / out-of-scope), find blind spots, run a devil's-advocate jury, generate a timed talk-track that cuts to fit the slot.
- **Portfolio aid** — draft project narratives the student rewrites, re-pitch for different audiences (hiring partner / planning board / lay reader), sequence and caption work.

## Design stance
- **Voice stays the student's.** It drafts and provokes; it never ships the final statement. Grade-the-trace: keep the drafts/edits visible.
- Accessibility-aware output (alt text, reading order) — crossover with **rap-tactile-cad** (a screen-reader-first portfolio).

## Likely build notes (candidates)
- LLM workflows + rubric-grounded prompts; static-site generator for the portfolio variant.
- Optional: ingest desk-crit transcripts to build the question forecast.
